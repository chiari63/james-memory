# 09_Reflections

Reflexões sobre a evolução da memória, do modo de trabalho do James e da qualidade do sistema.

Use esta pasta para aprendizados meta, como:

- quando criar ou não criar notas;
- problemas observados na organização;
- ajustes no protocolo de memória;
- padrões que reduzem entropia.

Não use esta pasta como diário emocional ou log de conversas.
