# 03_Projects

Projetos ativos, pausados, concluídos ou históricos.

Use esta pasta para esforços com objetivo, escopo e entregáveis. Projetos devem ter começo, direção e algum critério de conclusão ou mudança de fase.

Uma nota de projeto deve preferencialmente indicar:

- objetivo;
- status;
- decisões relevantes;
- entidades relacionadas;
- áreas impactadas;
- próximos ponteiros úteis, quando houver.

Evite transformar projetos em logs de execução. Marcos importantes pertencem a [[10_History/TIMELINE]].
