# 99_Archive

Itens inativos, obsoletos ou substituídos que ainda possuem valor histórico.

Arquivar não é deletar. Use esta pasta quando uma nota não deve mais aparecer como ativa, mas ainda pode explicar decisões, contexto passado ou evolução do sistema.

Ao arquivar uma nota:

- explique brevemente o motivo;
- marque `status: archived` se houver frontmatter;
- atualize links importantes;
- preserve apenas o que ainda tem valor futuro.
