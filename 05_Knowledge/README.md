# 05_Knowledge

Conhecimento consolidado e reutilizável.

Use esta pasta para sínteses, conceitos, decisões técnicas, explicações e aprendizados que podem ser úteis no futuro.

Boas notas nesta seção respondem perguntas como:

- o que sabemos sobre este assunto?
- qual decisão foi tomada e por quê?
- que problema difícil já foi compreendido?
- que padrão deve ser lembrado?

Não use esta pasta para logs brutos, respostas temporárias ou procedimentos passo a passo repetíveis. Procedimentos devem virar Skills.
